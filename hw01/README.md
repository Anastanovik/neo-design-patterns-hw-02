# Проєкт "Бібліотека" (TypeScript, ООП)

## Опис

Цей проєкт реалізує просту систему бібліотеки з використанням принципів об’єктно-орієнтованого програмування (ООП) у TypeScript.

Система моделює основні сутності:

- автори
- книги та електронні книги
- фізичні копії книг
- читачі
- бібліотека
- сервіс видачі книг

## Структура проєкту

/src
/models
Author.ts
Book.ts
EBook.ts
Copy.ts
Reader.ts
Library.ts
AbstractBook.ts
/services
BorrowService.ts
main.ts
tsconfig.json

## Використані принципи ООП

### Інкапсуляція

Поля класів приховані та доступні через методи.

### Наслідування

EBook наслідує AbstractBook.

### Абстракція

AbstractBook визначає базову структуру книги.

### Поліморфізм

Метод getDescription() реалізований по-різному в Book і EBook.

### Композиція

Library містить авторів, книги, копії та читачів.

## Команда запуску

npx ts-node src/main.ts
